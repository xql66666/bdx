# bdx
### 一个基于SpringCloud的北斗星工作室官网
##### 包含以下模块：
+    bdx_eureka          (注册中心)
+    bdx_user            (用户模块)
+    bdx_message         (留言模块)
+    bdx_info            (资讯活动模块)
+    bdx_source          (资源模块)
+    bdx_sms             (短信发送队列)
+    bdx_backstage       (后台模块)
+    bdx_zuul            (网关)
##### 由于配置中包含私人信息，上传时已忽略配置文件
